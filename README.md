MathUtil
========

From http://www.ziprealty.com/recruit/corporate.jsp :

In Java, create a class called MathUtil. Implement the following methods using only the '+' and '-' arithmetic operators (in other words, without using '*' or '/').
add(int a, int b)
e.g. add(1,2) => 3

subtract(int a, int b)
e.g. subtract(3,2) => 1

multiply(int a, int b)
e.g. multiply(2,4) => 8

divide(int num, int den)
e.g. divide(5,3) => 1 (Return a whole number; it is not necessary to round the result)

power(int num, int pow)
e.g. power(2,3) => 8


divide(int num, int den, int places)
e.g. divide(5,3,2) => 1.66 (Return a decimal number; it is not necessary to round the result)

powerRec(int num, int pow)
e.g. power(2,3) => 8 (Use recursion)
